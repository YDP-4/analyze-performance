### 컬러

- 배경, 기본 텍스트 컬러는 globals.css에 설정
- 커스텀 컬러는 기존 tailwindd 컬러랑 충돌되지 않도록 접두어 pf(performance)를 붙임

1. pf-purple: 버튼 또는 포인트 컬러 (Node 표시 등)
2. pf-lightgray: border, text
3. pf-gray: border, text
4. pf-navy: 컴포넌트 배경
5. pf-red: error
6. pf-green: success
7. pf-white: 흰색 필요한 곳
